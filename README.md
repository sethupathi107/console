This project involves building a CLI-based backend application that integrates with a weather API such as OpenWeather to fetch weather data for different cities. The application will allow users to enter commands in the terminal to retrieve current weather, view historical data, or filter results based on temperature or date. The system should fetch data in a paginated manner (if required), store it in a database, and avoid duplicate entries. Additional features like scheduled background updates and caching can be implemented to improve performance. This project helps students understand API integration, CLI input handling, and data storage without a web interface.
